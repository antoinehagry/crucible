# Crucible — Landing Page

Static landing page for **Crucible**, a systematic strategy-validation service for ES futures.

## Contents
- `index.html` — the full single-page site (self-contained, no build step)
- `Crucible Strategy Validation.pdf` — sample validation report, linked from the "View the full sample report" call-to-action

## Hosting
Served with **GitHub Pages** from the `main` branch (root). Live at:
https://antoinehagry.github.io/crucible/

To run locally, just open `index.html` in a browser.
